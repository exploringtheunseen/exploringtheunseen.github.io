# Exploring The Unseen
## Visualising India's Covid-19 Response

Our primary goals with this project are as follows: 
1.	Improve our understanding of data visualization: what are the theories, approaches, techniques, technologies and concepts that we can learn with respect to visualization? What are the best practices and break-through ideas is this field? 
2.	Improve our understanding of data analyses: how do we use data visualization to draw conclusions from data? To identify the outliers and anomalies? And how can this lead us to ask the right questions?
3.	Improve our data capabilities: how best to collect data from various sources and integrate them? What are the best (automated) workflows, and how do we roll our own for maximal automation?
4.	Make sense of what is happening in India with respect to Corona virus: Never waste a crisis! How can we apply the above three goals to the current situation and develop a comprehensive, nuanced picture of the spread and India’s response?
5.	Build a tool that helps the user explore the data as they see fit -- without having to write any code!
